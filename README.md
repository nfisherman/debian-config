# debian-config
my debian stuff

## Prerequisites
``git``

## Installing
```sh
wget -O- https://github.com/nfisherman/debian-config/releases/latest | unzip -
cd debian-config
sh install.sh
```

## Uninstalling
```sh
sh uninstall.sh

# skip uninstall confirmation
sh uninstall.sh -y
```
